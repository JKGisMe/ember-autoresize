Ember Autoresize

An ember-cli addon for autoresizing elements.

Installation

npm install --save-dev ember-autoresize
ember g ember-autoresize

Usage

To add autoresizing to an {{input}}, set autoresize to true:

{{input type="text" autoresize=true}}

You may then specify the maximum width that the input may resize to by setting the maxWidth property.

Example
<label for="pie-name">Pie Name</label>
{{input id="pie-name" value=name autoresize=true maxWidth=200}}<br><br>
<label for="pie-recipe">Recipe</label>
{{textarea id="pie-recipe" value=recipe autoresize=true maxHeight=200}}
